Данное руководство описывает процесс установки и запуска приложения на различных операционных системах.

Установка на Windows 10(версия сборки - не меньше 22h2)
1. Скачайте файл установщика с расширением .exe.
2. Запустите файл, дважды кликнув по нему.
3. Следуйте инструкциям установочного мастера.
4. По завершении установки на рабочем столе или в меню «Пуск» появится ярлык для запуска приложения.

Установка на macOS(версия Sonoma(14) и выше)
1. Скачайте файл с расширением .dmg.
2. Откройте скачанный .dmg двойным кликом.
3. В открывшемся окне перетащите иконку приложения в папку Applications.
4. Запустите приложение из папки Applications.

Установка на Linux(Ubuntu 24.04)
1. Скачайте файл с расширением .deb.
2. Откройте терминал и перейдите в папку с файлом, например: cd ~/Загрузки
3. Сделайте файл исполняемым: chmod +x vstrecheslav_x.y.z.deb, где x.y.z - номер версии приложения.
4. Запустите приложение: ./vstrecheslav_x.y.z.deb
Если используется .deb пакет, установите его командой:
sudo dpkg -i имя_файла.deb
sudo apt-get install -f
